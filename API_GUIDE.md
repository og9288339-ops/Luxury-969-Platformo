# API Endpoints Reference

## 1. Auth
- `POST /api/v1/users/signup` 
- `POST /api/v1/users/login` 
- `POST /api/v1/users/logout` 

## 2. Offers (P2P)
- `GET /api/v1/offers` 
- `POST /api/v1/offers` 
- `GET /api/v1/offers/:id` 

## 3. Escrow & Admin
- `POST /api/v1/escrow/create` 
- `PATCH /api/v1/admin/users/:id`
